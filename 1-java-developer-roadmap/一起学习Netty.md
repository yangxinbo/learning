# 一起学习Netty

### Netty是什么？

- Netty 是一个创建高性能网络应用程序的成熟的 IO 框架

- 相比较与直接使用底层的 Java IO API，你不需要先成为网络专家就可以基于 Netty 去构建复杂的网络应用。

### Netty能做什么？

- 业界常见的涉及到网络通信的相关中间件大部分基于 Netty 实现网络层。

### 为什么使用Netty 

- BIO vs NIO
- Java原生NIO API 从入门到放弃

### Netty 真实面貌

- Netty 中的重要概念及其关系
  - Eventloop
  - Boos、Worker
- There Model
- ChannelPipeline
- Pooling&reuse
- Native Transport
- 多路复用
